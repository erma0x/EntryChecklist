# Entry Confirmations Selector

A simple GUI application for selecting trading entry confirmations with python and tkinter.

![](./example.png)

## Description

This application provides a user-friendly interface with checkboxes to select different types of entry confirmations for trading strategies:

- **Change of structure**: Market structure shift indicator
- **Liquidity grab**: Liquidity capture pattern
- **Fake run**: False breakout detection
- **Sprint**: Rapid price movement confirmation

## Features

- Clean dark theme interface (black background with white text)
- Multiple checkbox selection
- Easy-to-use graphical interface
- Lightweight and fast

## Requirements

- Python 3.x
- tkinter (usually comes pre-installed with Python)

## Installation

1. Make sure you have Python installed on your system
2. Clone or download this repository
3. No additional dependencies required (tkinter is included with Python)

## Usage

Run the application with:

```bash
python app.py
```

Or on some systems:

```bash
python3 app.py
```

## How to Use

1. Launch the application
2. Select one or more confirmations by clicking the checkboxes
3. The selected confirmations will be tracked by the application
